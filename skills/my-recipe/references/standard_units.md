# 厨房标准单位换算参考 (Standard Culinary Units)

本文件作为 `my-recipe` 技能的基准，用于将用户的模糊单位换算为默认的 `g` 或 `ml`。

## 1. 默认单位 (Default Units)
- **固态配料**: `g` (克)
- **液态配料**: `ml` (毫升)

## 2. 常见中式厨房单位换算 (Typical Conversions)
| 用户单位 | 建议重量 (g/ml) | 备注 |
| :--- | :--- | :--- |
| 1 勺 (tbsp/大匙) | 15g / 15ml | 视调料密度略有不同 |
| 1 小勺 (tsp/小匙) | 5g / 5ml | 盐、糖等 |
| 1 碗 (标准饭碗) | 250g / 250ml | 约 250ml 水位 |
| 1 根 (葱/胡萝卜) | 50g - 150g | 建议根据大小动态估算 |
| 1 块 (姜/肉) | 20g - 500g | 建议询问具体克数 |
| 适量 / 少许 | 2g - 5g | 技能应提示用户具体的预估量 |

## 3. 单位记忆规则 (Preference Logic)
- 如果用户连续 3 次使用同一非标单位（如“2 瓷勺”），技能应记录并在后续清单中优先展示该单位，同时在括号内标注 `g`。
