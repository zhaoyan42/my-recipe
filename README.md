# My Recipe - 交互式烹饪助手技能 (Interactive Cooking Assistant Skill)

这是一个专为解决烹饪逻辑和采购麻烦而设计的 Gemini 技能。它不仅能提供食谱，还能陪你从选材、采购清单构建到最终的厨房实战。

## 核心特性 (Key Features)

- **交互式构建 (Construction Phase)**：
  当你决定做某道菜时，助手会通过一系列交互式对话，帮你规划出所需的**配料 (Ingredients)** 和 **厨具设备 (Equipment)**。它会问：“你家里已经有 XXX 了吗？”，从而为你剔除已备项。

- **动态采购核销 (Checkout Phase)**：
  这是此技能的灵魂。你可以分批次告诉它：“我买了红烧肉、冰糖和老抽”，它会立即响应并刷新你的**待办清单**，明确告诉你还差什么。

- **智能单位系统 (Smart Unit Logic)**：
  默认使用 `g` (克) 或 `ml` (毫升) 以保证烹饪的精准。同时，它会记录并学习你的用词偏好（如“3勺”、“2碗”），并在后续互动中优先使用你的习惯单位，同时在后台进行精准换算。

## 交互演示 (Demo)

### 阶段一：清单构建
**用户**: "我想做红烧肉。"
**助手**: "太棒了！红烧肉是一道经典菜。您打算做几人份？我来为您生成食材和厨具清单。顺便问一下，您厨房里现在已经有五花肉或老抽了吗？"
**用户**: "我有五花肉和砂锅了。"
**助手**: "收到！已从待办清单中剔除。以下是您还需要准备的：
- [ ] 冰糖 50g
- [ ] 老抽 20ml
- [ ] 八角 2个
- [ ] 炒锅 (用于最后收汁)
- [x] 五花肉 500g (已有)
- [x] 砂锅 (已有)"

### 阶段二：采购核销
**用户**: "我买了冰糖，老抽和八角。"
**助手**: "收到！已为您勾选。目前您的清单如下：
- [x] 冰糖 50g
- [x] 老抽 20ml
- [x] 八角 2个
- [ ] 炒锅 (还差这一样就齐了！)
- [x] 五花肉 500g (已有)
- [x] 砂锅 (已有)"
